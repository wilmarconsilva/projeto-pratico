# Projeto Prático
Projeto prático sobre versionamento semântico da aula de gerência de configuração de software.

O versionamento semântico é uma forma padronizada de numerar versões para indicar claramente o impacto da modificação execuada no software. O versionamento semântico contém três tipos de modificações: Major, Minor e Patch. Major são para alterações incopatíveis com a versão anterior, que modifica grande parte do software, Minor são novas alterações que implementam novas funcionalidades, Patch são alterações que corrigem as funcionalidades.
